Angular Pipes

1. Basics of Angular Pipes

Pipes in Angular transform data before displaying it in a view. Angular provides several built-in pipes, and you can also create custom pipes to suit your needs. Pipes can be classified into pure and impure based on their behavior.

2. Built-In Pipes

Angular comes with several built-in pipes, including:

  • DatePipe: Formats dates.
  • UpperCasePipe: Transforms text to uppercase.
  • LowerCasePipe: Transforms text to lowercase.
  • CurrencyPipe: Formats a number as currency.
  • DecimalPipe: Formats a number as decimal..
  • PercentPipe: Formats a number as percentage.
  • SlicePipe: Slices an array or string.
  • JsonPipe: Converts a value to JSON.

4. Pure and Impure Pipes

Pure Pipes: These are stateless and only depend on their input arguments. They are called only when Angular detects a pure change to the input value (primitive values change or object references change). By default, all pipes are pure.

Impure Pipes: These are called on every change detection cycle, regardless of whether the input arguments have changed. They can be useful for more complex transformations but should be used sparingly due to performance implications.
